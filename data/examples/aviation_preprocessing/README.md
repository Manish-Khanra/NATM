# Aviation Preprocessing Synthetic Example

This folder contains a tiny synthetic example for the aviation preprocessing workflow.

Files:

- `opensky_aircraft_db_sample.csv`
- `opensky_flightlists/flightlist_202201_sample.csv`
- `airports_sample.csv`
- `germany_calibration_input.csv`

The sample is intentionally small and only meant to illustrate the preprocessing steps and support
local tests/documentation. It is not intended to represent real OpenSky or official German totals.
